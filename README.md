# Hamad Ur Rehman — SQA Engineer Portfolio

A premium, modern QA Engineer portfolio built with Next.js and Tailwind CSS.

## Included

- Premium hero section
- Resume download button
- Featured project cards with screenshots
- CV-based experience, projects, achievements, skills, certifications, and education
- Contact form that opens the visitor's email app with the message prepared
- Responsive layout for desktop, tablet, and mobile

## Local Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Production Build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Keep the framework preset as Next.js.
4. Deploy.
